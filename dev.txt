# GOAL: Build a comprehensive, static 10-page website from a single topic input using ZERO backend or build tools.

## PHASE 1: INITIALIZATION & SCOPING
1. **Wait** for the user to provide a "Topic" or "Business Idea."
2. Once provided, perform a **Deep-Search** on the niche to identify:
   - Modern UI/UX trends for that industry.
   - The 10 most essential pages (e.g., index.html, features.html, pricing.html, about.html, contact.html, etc.).
   - **Mandatory Tech Stack:** Pure HTML5, Tailwind CSS (via CDN), Alpine.js (via CDN for state/interactivity), and Lucide Icons (via CDN).
   - **Image Sourcing:** Use free, contextually relevant images using Unsplash dynamic source links (e.g., `https://source.unsplash.com/featured/?{specific-keyword}`). Do not leave empty `src` tags.

## PHASE 2: ARCHITECTURAL BLUEPRINT
Before coding, generate a Markdown table mapping:
- Page Filename | Core Objective | Key Alpine.js Interactions | Image Keywords Needed.

## PHASE 3: AUTONOMOUS EXECUTION (STEP-BY-STEP)
You are now in "Anti-Gravity Agent Mode." Proceed through the following steps without further user intervention unless a critical error occurs:

### Step 1: The Global Blueprint (index.html setup)
- Create the foundational HTML boilerplate that will be used across all pages.
- **Strict Inclusions:** - `<script src="https://cdn.tailwindcss.com"></script>`
  - `<script> tailwind.config = { theme: { extend: { colors: { /* generated palette */ } } } } </script>`
  - `<script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>`
- Create a global `styles.css` for custom smooth-scrolling and glassmorphism utilities that Tailwind CDN doesn't cover natively.

### Step 2: Component Strategy (HTML Snippets)
- Design the core reusable UI blocks (to be copied into each HTML file):
  - **Responsive Navbar:** Must use Alpine.js (`x-data="{ open: false }"`) for the mobile menu toggle.
  - **Footer, Buttons, and Layout Wrappers.**

### Step 3: Sequential Page Generation (1-10)
- Generate the full HTML code for each of the 10 pages one by one.
- **Requirement:** Each page must feature high-quality "copywriting" (no Lorem Ipsum), fully responsive Tailwind grids, and proper SEO meta tags.
- **Interactivity:** Use Alpine.js (`x-data`, `x-show`, `x-transition`) to create "Anti-Gravity" UI elements—subtle fade-ins, expanding accordions for FAQs, and interactive pricing toggles.

## PHASE 4: FINAL INTEGRATION
- Ensure all pages are properly linked using standard relative HTML paths (e.g., `href="about.html"`).
- Perform a "Code Review" on yourself to ensure no broken links, missing closing tags, or broken CDN scripts.
- Provide a summary of the site architecture and instructions on how to open it directly in a browser.
